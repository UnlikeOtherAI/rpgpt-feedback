# RPGPT Feedback & Bug Reports

This is the public issue tracker for [RPGPT](https://rpgpt.unlikeotherai.com) — an AI-powered text RPG.

Use the **Report Error** button inside the app to submit reports automatically.  
You can also open an issue manually if you prefer.

## What to include

- What you were doing when the issue occurred
- What you expected vs. what happened
- Your device and OS version (filled in automatically by the app)

> **Note:** Reports submitted via the app are anonymized — only a random user ID, session ID, and device model are included. No personal information is collected.

## Labels

| Label | Meaning |
|---|---|
| `bug` | Something isn't working correctly |
| `feedback` | Gameplay suggestion or feedback |
| `user-report` | Submitted via in-app button |
